基于双向 LSTM 的共享单车短期需求预测

独立研究者 / 第一作者

独立撰写学术论文（国际会议在投）

技术栈：Python, PyTorch, Pandas, Scikit-learn

项目概述：针对城市共享单车需求波动问题，构建基于双层双向 LSTM 的短期需求预测模型。

数据与特征：使用深圳市政府开放数据平台约 100 万条订单记录（2017.04–2021.02），完成小时级数据聚合，引入周期性时间编码（sin/cos）、周末标记、早晚高峰标记等多维特征。

模型优化：设计多步输出策略避免误差累积放大；引入 Huber 损失函数替代 MSE 以提升对异常值的鲁棒性；将第一层 LSTM 改造为双向结构以捕捉前后时序依赖。进一步尝试随机森林融合模型降低残差。

实验结果：基线模型 MAE = 77.82, RMSE = 105.26；优化后 MAE 降至 61.45，RMSE 降至 91.61（提升 21.0% / 13.0%）；融合模型进一步降至 MAE = 55.50，RMSE = 85.55。独立撰写全英文学术论文（国际会议在投），为独立第一作者。
